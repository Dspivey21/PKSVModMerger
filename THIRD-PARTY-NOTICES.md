## Acknowledgments

The `data.trpfd` binary format was reverse-engineered by the Pokémon
modding community. Prior tooling that documented the format publicly:

- [Aqua-0/TrpfdUpdater](https://github.com/Aqua-0/TrpfdUpdater) — MIT licensed.
  This project's schema (`data.trpfd.fbs`) reflects the same on-disk layout
  TrpfdUpdater documents, but does not include any of its source code. Our
  C# accessors are independently generated by Google's `flatc` from our own
  `.fbs` file.

- [pkZukan/gftool](https://github.com/pkZukan/gftool) — documents the broader
  TRPFS/TRPFD virtual file system used by Game Freak titles.
